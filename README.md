## bronz Theme for oh-my-zsh

## Installation
Put <code>bronz.zsh-theme</code> in <code>.oh-my-zsh/custom/themes</code> directory
then in the <code>.zshrc</code> file set <code>ZSH_THEME="bronz"</code>
